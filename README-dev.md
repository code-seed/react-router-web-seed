## Setup
```
node -v
npm -v
npx create-react-app my-app
cd my-app
npm i react-router-dom
npm start
```

## Create directory and file
```
cd src
mkdir home
mkdir About
mkdir contact
mkdir no-page
mkdir nav-layout
cd home
touch Home.js
cd ..


```
